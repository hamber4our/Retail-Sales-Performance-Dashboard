# Retail-Sales-Performance-Dashboard

**Project Overview**

This project analyzes retail sales data using Microsoft Excel and presents key business metrics through an interactive dashboard. The goal was to transform raw sales data into actionable insights that support data-driven decision-making.

**Key Insights**
-Electronics generated the highest revenue among all categories.
-Laptop and Smartphone sales were the primary revenue drivers.
-Premium products contributed a significant share of overall revenue.
-Office Supplies recorded frequent transactions but lower revenue contribution.
-Promotional campaigns positively influenced customer purchasing behavior.
-Revenue was concentrated among a small number of high-value products.

**Conclusion**
This project demonstrates practical skills in Excel-based data analysis, dashboard development, KPI reporting, and business intelligence. It highlights the ability to convert raw data into meaningful insights that support strategic decision-making.
